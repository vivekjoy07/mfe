# mfe

### run container app
- `cd packages/container`
- replace ${NPM_TOKEN} in .npmrc file with Personal Access Token provided
- `npm i` (if first time)
- `npm start`

### run plp app
- `cd packages/plp`
- replace ${NPM_TOKEN} in .npmrc file with Personal Access Token provided
- `npm i` (if first time)
- `npm start`

### run pdp app
- `cd packages/pdp`
- replace ${NPM_TOKEN} in .npmrc file with Personal Access Token provided
- `npm i` (if first time)
- `npm start`
